## The Question

Every restaurant has reviews. Thousands of them. A manager can read twenty and get a feel for what customers think. But 6,166 reviews? You need a system that reads them all, finds the patterns, and tells you what actually matters.

This project builds that system three different ways — starting with simple frequency analysis and ending with unsupervised topic discovery — and asks the same question at each stage: what language, what phrases, what hidden themes predict whether a customer gives one star or five?

The answer turns out to be consistent across all three methods. And it is not what most restaurant owners assume.

<!-- METHODOLOGY -->
<table width="100%" cellspacing="0" cellpadding="0" style="border-left:3px solid #C9A84C;background:#0D1E35;margin:8px 0">
<tr><td style="padding:10px 16px">
<span style="font-family:monospace;font-size:10px;letter-spacing:3px;color:#C9A84C;text-transform:uppercase">How the Analysis Was Done</span>
</td></tr>
</table>

<br/>

Three methods are applied progressively to the same dataset, each answering the same question in a different way.

The first method counts. For each rating level, it finds which bigrams appear most exclusively — which phrases are disproportionately common in 1-star reviews and which are disproportionately common in 5-star reviews. This requires no model and produces immediately legible results.

The second method models. A LASSO logistic regression is trained on all 2,640 bigram counts with the Overall rating as the target. Rating 4 is relabelled as -1 to serve as the reference class. LASSO's penalty shrinks irrelevant bigrams to exactly zero, keeping only the phrases that genuinely discriminate between rating classes. This separates signal from noise across the full vocabulary.

The third method discovers. Latent Dirichlet Allocation treats each review as a mixture of hidden topics and finds 10 recurring themes without being told what to look for. Each review is then assigned a proportion for each topic, and a multinomial logistic regression of rating on those proportions shows which themes most strongly predict high versus low ratings.
